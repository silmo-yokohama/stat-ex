# STAT-EX デザインガイド

## 前提

このプロジェクトでのUI/UXに関連する全ての作業は`frontend-design`スキルを用いて、実装してください。

## デザイン方針

- **ライト基調**: 白背景 + チームカラーのダークグリーンをアクセントに使用
- **ダッシュボードUI**: グラフ・チャートを多用し、データを視覚的に訴求
- **カード + グリッドレイアウト**: 情報をカード単位で整理し、一覧性を確保
- **タイポグラフィ重視**: 数字やスコアを大きく太く印象的に表示。画像に頼らずデータとフォントで魅せるデザイン
- **一貫した配色ルール**: 勝敗・チーム表現を全ページで統一する

## タイポグラフィ

### フォント選定方針

- 実装時に frontend-design スキルを使用してクールなフォントを選定する
- 数字の視認性が最も重要（スタッツダッシュボードのため）
- 日本語と英数字でフォントを分けることを検討する
- Google Fonts から無料フォントを使用する

### サイズ・ウェイト方針

- スコア・大きな数字: 特大サイズ + Bold/Extra Bold
- Stats Cards の数値: 大サイズ + Bold
- テーブルの数値: 標準サイズ + Medium
- 本文テキスト: 標準サイズ + Regular
- ラベル・キャプション: 小サイズ + Regular

## グラフ（Recharts）

### 使用するグラフ種別

| グラフ | 用途 | 使用ページ |
|--------|------|-----------|
| 折れ線グラフ | 得点推移、スタッツ推移 | トップ、選手詳細 |
| グループ棒グラフ | Q別得点比較、月別成績 | 試合詳細、チーム成績 |
| 水平バーチャート | チーム比較、シューティングスプリット | 試合詳細、選手詳細 |
| エリアチャート | 累積勝利数 | チーム成績 |
| ドーナツチャート | 勝率（H/A比較） | チーム成績 |
| レーダーチャート | Q別得点傾向 | チーム成績 |
| バブルチャート/散布図 | 選手スタッツ分布 | 選手一覧 |
| カスタムバー | 勝敗ストリーク | 試合一覧 |
| プログレスバー | シューティング% | 選手詳細 |

### グラフの配色ルール

- 自チーム / 得点 / ポジティブ指標: `#006d3b`（ダークグリーン）
- 相手チーム / 失点 / ネガティブ指標: `#9CA3AF`（グレー）
- 理想ライン / 参考値: 点線で表示
- グリッド線: `#E6E7E7`（薄いグレー）
- チャートコンポーネントは汎用化し、異なるページで再利用可能にする

## レスポンシブデザイン

### ブレークポイント

| デバイス | 幅 | レイアウト |
|---------|-----|-----------|
| PC | 1024px〜 | 3〜4カラムグリッド |
| タブレット | 768px〜1023px | 2カラムグリッド |
| スマホ | 〜767px | 1カラム、スタック表示 |

### レスポンシブ方針

- モバイルとPC同等の優先度でデザインする
- テーブルはスマホで横スクロール対応
- チャートはスマホで縮小表示（タッチ操作対応）
- カードグリッドはスマホで1カラムにスタック

## コンポーネントパターン

### カード

- 角丸（rounded-lg）
- 薄いボーダー（border）
- 背景は白（bg-white）
- ホバー時に軽い影（hover:shadow-md）
- 勝ち試合カード: 左ボーダーにグリーンライン

### テーブル

- shadcn/ui の Table コンポーネントをベースにする
- ソート可能なカラムにはソートアイコンを表示
- スターター選手は太字で表示
- 横浜EXの行はプライマリライト背景でハイライト

### バッジ

- LIVE: レッド背景 + 白テキスト + パルスアニメーション
- FINAL: グレー背景 + 白テキスト
- WIN: グリーン背景 + 白テキスト
- LOSS: グレー背景 + 白テキスト
- HOME / AWAY: アウトラインスタイル

### ナビゲーション

- ヘッダー: サイトロゴ（STAT-EX）+ グローバルナビ
- ナビ項目: ダッシュボード / 試合 / 選手 / チーム / マスコット
- モバイル: ハンバーガーメニュー or ボトムナビ
- フッター: データ出典（B.LEAGUE公式・スポナビ）、最終データ更新日時

---

## OGP画像

- Next.js OG Image Generation を使用して動的OGP画像を自動生成する
- ページごとにスタッツ情報を含むOG画像を生成
- 背景: ダークグリーン、テキスト: 白
- ページタイトル例: 「横浜エクセレンス vs 愛媛オレンジバイキングス 試合結果 - STAT-EX」

---

## アクセシビリティ

- WCAG 2.1 AA 準拠（カラーコントラスト 4.5:1 以上）
- チャート・テーブルに適切な aria 属性を設定する
- キーボードナビゲーション対応（タブ操作で全機能にアクセス可能）
- 画像には必ず alt テキストを設定する
